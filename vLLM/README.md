# vLLM 기반 퀴즈 생성 및 자동 번역 로직

## 개요
로컬 호스팅된 LLM(vLLM 서버)을 통해 영문 퀴즈를 생성한 후, 별도 준비한 한국어 번역 전용 모델을 이용해 퀴즈 내용을 한국어로 자동 번역 및 후처리하는 파이프라인 프로젝트입니다.

## 주요 기능
- **퀴즈 생성**: `requests` 모듈로 로컬 vLLM 서버의 엔드포인트(`http://localhost:8000/v1/chat/completions`)에 특정 생성 주제를 던져, 영문 객관식 퀴즈(문항, 보기, 정답, 해설)를 구성하게 합니다.
- **언어 번역 처리**: Hugging Face `transformers` 라이브러리를 통해 로컬 경로에 존재하는 파인튜닝 번역 모델(`Llama-3.2-3B-KO-EN-Translation`)을 로드하고, 산출된 영문 퀴즈를 한국어 맥락에 맞게 변환합니다.
- **텍스트 후처리**: 생성 및 번역된 텍스트 결과에서 불필요한 태그나 기호 등을 스크립트 차원에서 정제하여 최적의 단문 포맷으로 제공합니다 (`clean_text`).

## 실행 방법
1. vLLM 서버를 별도로 띄워 URL 통신이 가능토록 대기 상태로 이행합니다.
2. 메인 스크립트 내에서 주제(예: "Computer Science")를 설정 후 다음 명령어를 실행합니다.
```bash
python model.py
```
